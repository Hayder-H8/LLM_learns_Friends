# LLM_learns_Friends
Small LLM built from scratch to learn F.R.I.E.N.D.S series script

Tokenize is costumize ti a part of the script to pick up on several key tokens like caracter names and catch phrases if any 


There is the possibility to train the model from scratch , pick up from a ckpt point or do inference directly based on a saved ckpt 


There is the possibility to adjust several parameters related to the model , the training  , and inference
